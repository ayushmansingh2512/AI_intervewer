# Solvithem Frontend

This is the frontend component of the Solvithem AI Interviewer platform, built with React, Vite, and Tailwind CSS.

For full project documentation, setup instructions, and features, please refer to the [Root README](../README.md).

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```
